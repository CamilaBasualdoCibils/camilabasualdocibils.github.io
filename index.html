AtlasNet

Worlds Without
Compromise

Run vast online worlds with automatic orchestration, spatial sharding, and seamless player movement across servers.

Features

Let me tell you what this baby can do

Dynamic Sharding

Scale worlds automatically as players move and populations change—no manual sharding logic required.

Multiple Worlds

Run multiple independent worlds, realms, or simulations—each with its own coordinate system and scaling strategy.

No Engine Lock-In

AtlasNet provides bindings and SDKs for major languages and game engines, so you can integrate it directly into any stack.

Seamless Transitions.
Invisible Boundaries.

AtlasNet ensures players traverse a distributed world as if it were a single continuous space.

Not Cartesian? Not a Problem!

Built-in support for Cartesian and geospatial coordinate systems, with seamless world partitioning across them all.

Web Interface

Cartograph provides a unified web-based control plane for monitoring, managing, and operating your entire AtlasNet deployment in real time.

One Size Doesn't Fit All

Different worlds require different partitioning models. AtlasNet includes multiple models including grid-based, KD-tree, Voronoi, and AI-assisted heuristics to match your simulation’s needs.
